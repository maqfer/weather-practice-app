# Weather App (Прогноз погоды)

Учебный проект для сдачи практики по веб-разработке. Простое веб-приложение для получения актуальной информации о погоде в любом городе мира.

## Функционал

- Поиск погоды по названию города
- Отображение температуры, влажности и скорости ветра
- Отображение иконки и описания погодных условий на русском языке
- Защита от добавления дубликатов городов
- Адаптивный дизайн для мобильных устройств

## Стек технологий

- **Frontend:** HTML5, CSS3, JavaScript (ES6, Async/Await)
- **API:** OpenWeatherMap API
- **Шрифты:** Google Fonts (Roboto)

## Как запустить локально

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ТВОЙ_НИК/weather-app.git
2. Откройте файл index.html в браузере (двойным кликом)
3. Проект использует временный тестовый API-ключ. При необходимости замените его на свой в файле js/config.js

## Использование

1. Введите название города в поле поиска
2. Нажмите кнопку "Поиск" или клавишу Enter
3. Карточка с погодой добавится на страницу

## Примечание

В проекте используется временный тестовый API-ключ OpenWeatherMap для упрощения проверки. В реальном проекте необходимо получить собственный ключ на сайте OpenWeatherMap(https://openweathermap.org/api).
